# 需求结构

```text
    ├─ 用户登录
    │  ├─ 保持登录，通过token验证实现
    │  ├─ 用户注册
    │  │  ├─ 注册成功后，自动返回登录页或者登录到首页
    ├─ 首页
    │  ├─ 编辑簿
    │  ├─ 阅读器
    │  ├─ 论坛社区
    │  ├─ 关于开发者
    │  ├─ 用户设置
    │  │  ├─ 个人中心
    │  │  │  ├─ 我的信息
    │  │  │  │  ├─ 信息展示 [√]
    │  │  │  │  ├─ 信息编辑 [√]
    │  │  │  │  ├─ 修改密码 [√]
    │  │  ├─ 用户登出 [√]
    │  │─ 我的组织
    │  │  ├─ 列表
    │  │  │  ├─ 我创建的群组
    │  │  │  │  ├─ 编辑/解散/踢人/邀请/成员列表/成员权限管理
    │  │  │  ├─ 我加入的群组
    │  │  │  │  ├─ 退出/成员列表
    │  │  ├─ 新建/编辑（可邀请检索到的成员）
    │  │  ├─ 解散（删除）
    │  │  ├─ 检索
    │  │  │  ├─ 我创建的和我加入的 - 进行标记/查看详情
    │  │  │  ├─ 未知群组 - 申请加入/查看详情
    │  ├─ 我的信箱
    │  │  ├─ 验证信息 [√]
    │  │  ├─ 系统公告（用户接收到的公告，只读）[√]
    │  │  ├─ 用户反馈 [√]
    └─ 
```
